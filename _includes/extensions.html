Extensions

  • IpSensorMan - exposes heart rate, cycling speed and cadence.

  • Music player - controls music player and exposes information like current track name, author and album name.

  • Tasker - allows to run a task defined in Tasker.

  • Weather - provides weather conditions and 5-day forecast for the given location and language (temperature, pressure, humidity, wind speed, description and icon code).

  • Sample - sample extension with available source code.
